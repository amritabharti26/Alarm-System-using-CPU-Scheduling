# CPU Scheduler & Reminder System

An enhanced Android application that visualizes various CPU scheduling algorithms and integrates them into a real-time reminder/alarm system.

## 📥 Download & Try
You can download the latest installation file (APK) from the link below:

[🚀 **Download app-debug.apk**](https://github.com/amritabharti26/FCFSReminderScheduler/releases/latest)

## 🚀 Features
- **Algorithm Selection**: Choose between FCFS, SJF, Priority, and Round Robin.
- **Dynamic Inputs**: Input fields adapt based on the selected algorithm (e.g., Priority or Time Quantum).
- **Sequential Execution**: Watch tasks execute one-by-one with a live alarm and progress bar.
- **Visual Results**: Real-time calculation of Waiting Time, Turnaround Time, and averages.
- **Modern UI**: Clean, dark-themed interface with rounded cards and intuitive controls.

## 🛠 Scheduling Algorithms Included
1. **FCFS (First Come First Serve)**: Simple non-preemptive scheduling based on arrival time.
2. **SJF (Shortest Job First)**: Prioritizes tasks with the shortest duration to minimize average waiting time.
3. **Priority Scheduling**: Executes tasks based on assigned priority levels.
4. **Round Robin (RR)**: Time-sharing execution using a fixed Time Quantum.

## ⚙️ How to Run
1. Clone the repository.
2. Open in Android Studio (Koala or newer).
3. Build and run on an emulator or physical device (Min SDK 24).
